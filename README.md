# рҹ“Ұ Installation

Clone the project:

```bash
git clone https://github.com/Tarek-Hegazy/Admin-Dashboard---Products-Categories.git
```

Go to project directory:

```bash
cd Admin-Dashboard---Products-Categories
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```
